# Online Job Portal System

## Description

A full-stack Job Portal web application developed using Spring Boot and Angular. The platform connects job seekers and recruiters, allowing recruiters to post and manage jobs while job seekers can search jobs, upload resumes, and apply online. The system includes JWT authentication, role-based authorization, recruiter approval workflow, job status tracking (active/closed jobs), and applied-job indicators for users who have already submitted applications.

## Features

- JWT Authentication and Authorization
- Role-based Access Control
- Recruiter Approval by Admin
- Job Posting and Management
- Job Search Functionality
- Resume Upload and Download
- User Management
- Secure REST APIs
